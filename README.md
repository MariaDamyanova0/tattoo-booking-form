# Tattoo Booking Request

A modern, user-friendly booking request form for a tattoo studio.  
Built as a front-end practice project with a focus on clean UI, validation, and local data persistence.

## Live Demo
https://mariadamyanova0.github.io/tattoo-booking-form/

## Features
- Booking request form with validation  
- Inline error messages for required fields  
- Save requests to localStorage (persists on refresh)  
- Requests management panel:
  - Mark as contacted  
  - Delete request (with confirmation)  
  - Clear all requests  
  - Search/filter requests  
- Responsive layout  
- Dark studio-themed UI with hero image  

## Tech Stack
- HTML  
- CSS  
- JavaScript (Vanilla)  
- GitHub Pages (deployment)
